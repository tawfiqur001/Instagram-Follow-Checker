# Instagram Follow Checker

A simple web-based tool to analyze Instagram followers and following data exported from Instagram. It helps you quickly find:

* Who doesn’t follow you back
* Your followers (“fans”)
* Mutual followers

---

## Features

* Upload multiple `followers.json` files
* Upload `following.json` file
* Instantly compare datasets
* View:

  * Followers count
  * Following count
  * Not following back
  * Mutual followers
* Search usernames
* Copy results with one click
* Fully offline (runs in browser)

---

## How to Use

1. Export your Instagram data from Instagram settings
2. Extract the files
3. Upload:

   * All `followers_*.json` files
   * Your `following.json` file
4. Click **Analyze Account**
5. View results instantly
